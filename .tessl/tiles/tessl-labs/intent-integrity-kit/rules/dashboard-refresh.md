# Dashboard Refresh

After completing any action that creates or modifies files in `specs/` or project root (`CONSTITUTION.md`, `PREMISE.md`), regenerate the dashboard:

```bash
bash .tessl/tiles/tessl-labs/intent-integrity-kit/skills/iikit-core/scripts/bash/generate-dashboard-safe.sh
```

Windows: `pwsh .tessl/tiles/tessl-labs/intent-integrity-kit/skills/iikit-core/scripts/powershell/generate-dashboard-safe.ps1`
